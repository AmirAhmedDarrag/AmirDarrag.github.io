Night At The Museum Project

What this project is? a Virtual Reality game as project of Udacity VR Nanodegree.The idea is to build a museum for VR artifacts (i.e devices, companies, projects). First, I built a 3D model of the museum. Then I placed it into the scene with unity. I then started by building The UI, the first element is a help text explaining how to navigate the museum. Then a waypoint leading to 5 different stations is presented. each station contains a VR artifact, a text describing it, and a voice reading the text. at first, there was an issue with the narrator where the voice doesn't stop once you leave the stations. Ahmed kamal, A User who tried the application described it as "overlapping voices of narrators at different stations". The problem was overcome by modifying the code controlling the waypoint, by adding logic to stop the narrating voice once the user leaves the station.
